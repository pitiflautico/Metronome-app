# Sound Files

Place the following MP3 files in this directory:

- `click.mp3` - Classic metronome click sound
- `ping.mp3` - High-pitched ping sound
- `stick.mp3` - Drumstick click sound
- `wooden.mp3` - Wood block sound
- `drum.mp3` - Drum hit sound

You can create these sounds using:
- Online metronome sound generators
- Audio editing software (Audacity, GarageBand, etc.)
- Free sound libraries (Freesound.org, etc.)

Recommended specifications:
- Format: MP3
- Sample rate: 44.1 kHz
- Bit rate: 128-192 kbps
- Duration: 50-200ms
- Mono or Stereo

For testing, you can use any short click/percussion sounds.
